# Tiny STL

C++ 课程作业，实现一个 STL。使用 xmake 构建，~~cmake 我学不会~~。

## Build && Run

关于安装 xmake，请参考其[官方文档](https://xmake.io/#/zh-cn/guide/installation)。

以 Ubuntu 20.04 为例。

```bash
# install xmake
bash <(wget https://xmake.io/shget.text -O -)
# build
xmake
# run
xmake run
```
